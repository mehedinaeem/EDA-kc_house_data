# EDA (KC_house_data)

This project contains exploratory data analysis (EDA) for the King County House Sales dataset.

## Folder Structure

```
EDA (KC_house_data)/
│
├── data/
│   └── kc_house_data.csv
│
├── notebooks/
│   └── 01_eda.ipynb
│
├── scripts/
│   └── main.py
│
├── outputs/
│   └── figures/
│
└── README.md
```

- **data/**: Contains the dataset.
- **notebooks/**: Jupyter notebooks for EDA and analysis.
- **scripts/**: Python scripts for data processing and analysis.
- **outputs/figures/**: Generated figures and plots.

## Dataset
The dataset is from King County, USA, and includes house sales data.

## Usage
- Run scripts in the `scripts/` folder for data processing.
- Use the notebook in `notebooks/` for interactive EDA.
